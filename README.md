## Ruby on Rails version

Ruby '2.5.1'

Rails '5.2.0'


## Heroku : https://nasa-security.herokuapp.com/


## Steps:
```
$ bundle install --without production
```
```
$ rails db:migrate
```
```
$ rails server
```

## Open your browser and launch
```
$ localhost:3000
```

## If it doesn't work
```
$ rails server -p 4567
```
## Launch
```
$ localhost:4567
```
